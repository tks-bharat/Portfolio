# **App Name**: Profolio Canvas

## Core Features:

- About Me Section: Dedicated space to introduce yourself, highlighting key experiences and personal information.
- Interactive Project Gallery: Visually engaging showcase of your work with case studies to demonstrate project impact.
- Skills Showcase: Highlight your expertise with interactive elements to display proficiency levels.
- CV Download: Direct link for visitors to download your resume for offline viewing and application.
- AI-Powered Content Generation: Generative AI tool to automatically create tailored summaries or introductions for each section of the portfolio using provided details, acting as a tool for overcoming writer's block.

## Style Guidelines:

- Primary color: Deep slate blue (#2c3e50) to convey professionalism and trustworthiness.
- Background color: Light cool gray (#f0f4f8) to create a minimalist backdrop.
- Accent color: Soft sky blue (#77b5fe) for interactive elements and highlights.
- Body and headline font: 'Inter' sans-serif for clean readability and a modern aesthetic.
- Simple line icons to complement the minimalist style; icons used for skills and contact information.
- Clean, grid-based layout to ensure content is well-organized and easy to navigate.
- Subtle transitions and hover effects to add interactivity without overwhelming the user.
